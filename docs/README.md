# Fellowship Pack v1.0.0

Cognitive Environment Layer for macOS & iPadOS.

## Environments

| Environment | Cognitive State | Character |
|---|---|---|
| Signal · Day | Receptive learning | Calm, mid contrast — references surface |
| Signal · Moonlight | Receptive, dimmed | Same calm, lower light — the beam narrows |
| Drift | Light arousal | High luminance, open gradients, associative |
| Forge | Hyperfocus | Low luminance, single narrow highlight |
| Hearth | Sustained attention | Neutral mid tones, gentle contrast |
| Havens · Day | Sanctuary / idle | Luminous, open, asking nothing |
| Havens · Moonlight | Closure | Cool dim glow — enough to see by |

## Technical Guarantees

- sRGB IEC61966-2.1 embedded
- 16-bit master source files recommended
- Dithered 8-bit final exports
- Gradient banding inspected
- Night-room validation recommended (low-light environments at ~30% display brightness)

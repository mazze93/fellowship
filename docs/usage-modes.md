## Circadian Routing

Fellowship transitions environments automatically based on time-of-day, schedule context,
and workflow signals. The circadian router debounces competing signals and resolves them
into one coherent environment transition — wallpaper, focus mode, notification posture,
and surfaced tools move together as a single coordinated shift.

## Nervous System Targets

Each environment is calibrated to a specific arousal state and tuned against its
characteristic failure mode:

| Environment | Target State | Tuned Against |
|---|---|---|
| Signal · Day | Parasympathetic calm | Drowsiness |
| Signal · Moonlight | Parasympathetic calm | Drowsiness |
| Drift | Light arousal | Eye fatigue |
| Forge | Hyperfocus | Tunnel collapse |
| Hearth | Sustained attention | Visual monotony |
| Havens · Day | Idle / restorative | Over-softening |
| Havens · Moonlight | Closure | Overstimulating highlights |
